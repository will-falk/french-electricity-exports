This is a project to expand my Pandas skills and learn how to work with electricity data. So far, I've pulled data on Norwegian and French cross-border physical flows from ENTSO-E.

I wrote the Python that extracts and processes the ENTSO-E data myself (via the [entsoe-py client](https://github.com/EnergieID/entsoe-py/tree/master)), used Claude Code to generate a basic Dash app to visualize the data, and deployed the app via [Render](https://render.com/).

You can view the live visualization here:
https://french-electricity-exports.onrender.com/
