This is a project (very much a work in progress) to expand my Pandas skills and learn how to work with electricity data. So far, I've pulled data on Norwegian and French cross-border physical flows from ENTSO-E.

I wrote the Python that extracts and processes the ENTSO-E data myself (via the [entsoe-py client](https://github.com/EnergieID/entsoe-py/tree/master), used Claude Code to generate a basic dash app to display it, and Render to publish it.

You can view the live visualization here:
https://french-electricity-exports.onrender.com/
